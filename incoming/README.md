# Drop images here

Put any pictures you want added to the website in this folder. Then tell
Claude "check the incoming folder" and it will review each image, decide the
best placement, optimise it, move it into the site, and remove it from here.

## How to help me place them well

- **Optional hints in the filename** make placement more accurate, e.g.
  - `hero-factory.jpg`      → homepage hero slideshow
  - `banner-military.jpg`   → Military page banner
  - `card-railway.jpg`      → homepage Railway capability card
  - `about-team.jpg`        → About page
  - `logo-acme.png`         → a principal/client logo
- No hint is fine too — I'll judge from the picture itself and ask if unsure.

## Rough size guidance (not strict)

- Hero / page banners: large & landscape, ~2000px+ wide
- Capability cards / thumbnails: ~1200px wide
- Logos: transparent PNG preferred

Anything works though — I'll resize/convert as needed.
